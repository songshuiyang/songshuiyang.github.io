4. 使用Jenkins

4.1 配置工具环境配置

上面的环境都准备好了之后，先配置一下 参数(系统管理-> 全局工具配置) git不用配置使用默认

4.2 创建一个任务

1.点击创建任务，如果在创建项目时候，没有“创建一个Maven 项目”的选项。你需要安装Maven项目插件：Maven Integration plugin 。

2.配置源码管理，jenkins会自动拉取代码放到/var/lib/jenkins/workspace/任务名 路径下

3.配置Maven构建，构建完后会自动打包

4.编写shell脚本，maven构建完成之后就是把打好的包放在tomcat下了并启动
