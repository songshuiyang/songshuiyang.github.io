前言

  • 本系列是集合系列的最后一个专题，为什么放到最后，因为Set的子类HashSet、LinkedHashSet底层是基于HashMap来实现的

  • Set接口代表的元素是无序、不可重复的集合，是 Collection的三大金刚之一，这一专题系列将介绍集合Set子类的实现

  • 再来回顾下集合框架图，可以看到Set主要有下面这些实现类，感觉和Map接口的子类一毛一样

    • HashSet 基于哈希值的无序不可重复集合
    • LinkedHashSet 基于哈希值的有序不可重复集合
    • TreeSet 可自定义排序规则的不可重复集合
