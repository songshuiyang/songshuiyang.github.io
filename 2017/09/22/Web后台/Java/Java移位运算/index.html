Java移位运算

2017-09-22
  • server

<< 是与>>对应的左移运算符，表示将exp1向左移动exp2位，在低位补0。其实，向左移动n位，就相当于乘以2^n。

>>> : 无符号右移

res = 20 >>> 2; 的结果与 >> 相同；

res = -20 >> 2;

-20的二进制为 1110 1011，右移2位，此时高位补0，即 0011 1010，结果为 res = 58;

也叫逻辑右移，即若该数为正，则高位补0，而若该数为负数，则右移后高位同样补0。
