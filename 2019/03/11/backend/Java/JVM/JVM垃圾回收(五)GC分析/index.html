GC日志查看

  • 可以通过在java命令种加入参数来指定对应的gc类型，打印gc日志信息并输出至文件等策略。GC的日志是以替换的方式(>)写入的，而不是追加(>>)，如果下次写入到同一个文件中的话，以前的GC内容会被清空。

  • 对应的参数列表

  • 示例 -XX:+PrintGCDateStamps -XX:+PrintGCDetails -Xloggc:./gclogs
