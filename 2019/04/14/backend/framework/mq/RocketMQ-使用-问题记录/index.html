    RocketMQ发送消息时不能自动创建topic: No route info of this topic

    • 解决方法
      • 1、启动broker时加上autoCreateTopicEnable=true, Windows下的命令
        • start mqbroker.cmd -n 127.0.0.1:9876 autoCreateTopicEnable=true
        1. rocketmq服务端的版本和客户端的版本需要保持一致
          • maven
