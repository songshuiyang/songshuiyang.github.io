前言

什么是Spring Cloud Ribbon

  • Ribbon是为客户端提供负载均衡功能的服务

  • 百度百科：

    • 英文名称为Load Balance
    • 其含义就是指将负载（工作任务）进行平衡、分摊到多个操作单元上进行运行
